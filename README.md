# Home InfoScreen (Qt / QML)

A lightweight, always-on home information screen built with **Qt / QML**.  
The application provides essential daily information at a single glance, such as **weather**, **public transport schedules**, and **calendar events**.

Designed for **Raspberry Pi + display**, but can also be used on other always-on screens such as **tablets (e.g. iPad)** or wall-mounted monitors.
